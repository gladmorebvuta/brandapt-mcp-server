# BrandaptOS MCP Server

Gives Claude live access to your BrandaptOS platform from any Claude tab (Chat, Cowork, or Code).

## What It Does

Once installed, Claude can:
- Read and update all ventures with status, health, and stage
- Browse and search VentureChat conversation history
- Log decisions, milestones, and strategic insights to venture project logs
- Search across all conversations to find past reasoning
- Check venture analytics metrics (card taps, signups, etc.)
- See what documents are in the knowledge base

## Setup

### Step 1 — Build the server

```powershell
cd C:\Dev\BrandApt\brandapt-mcp-server
npm install
npm run build
```

### Step 2 — Get a Firebase service account key

1. Go to [Firebase Console](https://console.firebase.google.com) → Project: `brandaptos-v2`
2. Click the gear icon → **Project Settings**
3. Click **Service Accounts** tab
4. Click **Generate new private key** → Save as `brandapt-service-account.json`
5. Put the file inside the mcp-server folder:
   `C:\Dev\BrandApt\brandapt-mcp-server\brandapt-service-account.json`

   ⚠️ NEVER commit this file to git — add it to `.gitignore` immediately.

### Step 3 — Add to Claude Desktop App

Open Claude desktop app → **Settings** → **Developer** → **MCP Servers**, and add:

```json
{
  "brandapt": {
    "command": "node",
    "args": ["C:/Dev/BrandApt/brandapt-mcp-server/dist/index.js"],
    "env": {
      "BRANDAPT_SERVICE_ACCOUNT_PATH": "C:/Dev/BrandApt/brandapt-mcp-server/brandapt-service-account.json",
      "BRANDAPT_PROJECT_ID": "brandaptos-v2"
    }
  }
}
```

### Step 4 — Restart Claude

Close and reopen the Claude desktop app. You should see BrandaptOS tools available.

## Available Tools

| Tool | What it does |
|------|-------------|
| `brandapt_list_ventures` | List all ventures with status and health |
| `brandapt_get_venture` | Get venture details + recent project log |
| `brandapt_list_logs` | Browse decision/milestone log for a venture |
| `brandapt_add_log_entry` | Add a decision, milestone, or insight to a venture's log |
| `brandapt_list_conversations` | Browse VentureChat conversations |
| `brandapt_get_conversation` | Read a full conversation with messages |
| `brandapt_create_conversation` | Start a new VentureChat conversation |
| `brandapt_add_message` | Add a message to a conversation |
| `brandapt_search_conversations` | Search conversations by keyword |
| `brandapt_list_documents` | See what's in the RAG knowledge base |
| `brandapt_get_venture_metrics` | Get analytics metrics for a venture |
| `brandapt_get_global_metrics` | Get overall AI feedback stats |
